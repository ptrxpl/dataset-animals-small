# dataset-animals-small

Source: https://images.cv/

After downloading, the photos were reviewed and selected (there are no "strange" samples).

Size: 512x512 px

Folder structure:
```
├── 📁animals                 
│   ├── 📁bear     # 375 photos
│   ├── 📁fox      # 245 photos 
│   ├── 📁seals    # 252 photos 
│   ├── 📁shark    # 282 photos 
│   └── 📁zebra    # 470 photos 
```
Feel free to use this dataset in e.g. CNN (Convolutional Neural Network) toy problems.
